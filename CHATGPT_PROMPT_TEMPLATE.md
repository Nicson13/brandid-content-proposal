# ChatGPT 标准 Prompt 模板
# Brandid · Phase 01 · 选题内容生成

---

## 使用方式
1. 将 BrandDNA Collector 导出的 brand.json 内容复制粘贴到 [BRAND_JSON] 位置
2. 将完整 Prompt 发送给 ChatGPT（GPT-4 或以上版本）
3. ChatGPT 输出纯 JSON → 保存为 content.json
4. 将 content.json 导入 Proposal Builder

---

## Prompt 正文（直接复制使用）

```
你是一位资深的社交媒体内容策略师，专门为品牌制定高效的内容选题方案。

以下是客户的品牌资料（JSON格式）：

[BRAND_JSON]

---

请基于以上品牌资料，生成一份社交媒体内容选题方案。

**要求：**
- 生成 6–8 个选题方向
- 每个选题需包含：标题、目标（曝光/互动/转化）、受众、内容格式、开场钩子、文案参考
- 内容格式只能选：Reels、Carousel、Graphic Poster、Story
- 选题要多样化，覆盖不同目标和格式
- 文案参考需真实可用，风格符合品牌口吻
- 所有选题需符合品牌个性和目标受众

**输出格式要求：**
- 只输出纯 JSON，不要加任何说明文字
- 不要用 markdown 代码块包裹（不要 ```json）
- 严格按照以下 JSON 结构输出：

{
  "summary": "（用1-2句话概述整体内容策略方向）",
  "topics": [
    {
      "id": 1,
      "title": "（选题标题，吸引眼球，10-25字）",
      "goal": "曝光",
      "audience": "（目标受众描述，1句话）",
      "format": "Reels",
      "hook": "（开场钩子，前3秒抓住注意力的内容，20-40字）",
      "caption": "（正文文案参考，包含 CTA，80-150字）"
    }
  ],
  "toneNote": "（品牌口吻使用指南，2-3句话）",
  "nextSteps": [
    "（下一步行动建议1）",
    "（下一步行动建议2）",
    "（下一步行动建议3）"
  ]
}
```
